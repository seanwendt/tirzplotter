TirzPlotter Privacy Policy

Last updated: April 27, 2026

TirzPlotter is an educational pharmacokinetic visualization tool. It does not provide medical advice and should not be used to make treatment decisions.

Data You Enter

Injection dates, dose amounts, and body weight values are stored locally in your browser or app WebView storage. TirzPlotter does not send this dose schedule to a TirzPlotter server.

Export and Import

If you use Export, TirzPlotter creates a JSON backup file on your device. If you use Import, TirzPlotter reads the JSON file you select and stores it locally in the app. You control those files.

Hosting and Third Parties

The web app is hosted by Netlify, which may process standard request information such as IP address, user agent, and request logs as part of operating the site. The app may load third-party resources such as charting libraries, fonts, and donation-button assets.

No Accounts

TirzPlotter currently does not require account creation, Google login, or cloud sync.

Contact

For privacy questions, use the support contact listed on the Google Play Store listing or the project repository.
